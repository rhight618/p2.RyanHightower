B: Microwave Usage and Proposed Design

Common Microwave Uses

  1. Cooking Food
  2. Reheating Food
  3. Boiling Water
  4. Defrosting Frozen Food

The inteface makes these common uses cases realtively easy. Most require time-based heating which can be accomplished with simple time settings. The interface falls a little short in having proper timing assocatiated with common food types like potatoes, boiling water.

Rare Microwaves Uses

  1. Heating beauty products
  2. De-crystalization science experiments

Common Sequences of Actions

  1. Place food in oven -> Close Door -> Press Time Cook -> Enter in Time on number pad -> Press Start
  2. Place food in oven -> Close Door -> Press one of the Express Cook Numbers to cook for the corresponding minutes
  3. Place food in oven -> Close Door -> Press one of the Express Cook Numbers to cook for the corresponding minutes -> Add 30 secs by pressing the +30sec buttons

User Support and Feedback

  1. This microwave makes audible beeps when buttons are pressed
  2. When the cooking time is complete multiple beeps are sounded to notify the user
  3. A display shows the current time left in the cooking cycle as well as when a time is being typed in
  4. The display could be more informative. There are instructions inside the door of the microwave, but if you dont kow they are there, they arent any help

Common Mistakes Made

  1. Forgetting that buttons 1-6 are Express Cooking Presets. You must click Time Cook first for setting specific time
  2. Assuming that the Popcorn preset will cook for the exact correct amount of time. If the bag is smaller you will likely scorch the popcorn

Suggested Improvements to Design

  1. Eliminate the Sensor Cook Menu all together because the sesnor cooking is not accurate and leaves the food undercooked/overcooked
  2. Emiminate the "Soften" Picture. It is confusing to the user as to what he/she is supposed to do to use it. It is not a button but merely instructions
  3. Add a section for Programable Preset buttons. The user will be able to add a custom name and set the cooking time. Once this is saved, the user will be able to press the button for express cooking
  4. Remove the +30 button. It is handy for adding extra time, but now the user can add a preset for that capability
  5. Add a section with a selectable alphabet so the user can for set a Preset Button name
  6. Deactivate and grey out buttons that are not selectable during preset sequences
  7. Add a readout display that gives the user better feedback and guidance while using the microwave
